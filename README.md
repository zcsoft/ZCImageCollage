# ZCImageCollage

Image Collage


把多张图片合并成一张图片，可用于给图片添加边框，标签，贴纸等。或者拼接多张图片成一张。


使用方法：

添加 ZCImageCollage.h ZCImageCollage.m 到工程中，并在需要使用的位置#import "ZCImageCollage.h"即可。
